# FuckUpdate

## 简单介绍
FuckUpdate是一个整蛊软件，他能让您的电脑“蓝屏”，然后您就可以去愉快的摸鱼了。

## 使用方法

### Step 1

将浏览器设置为全屏。如果你使用的是Chrome浏览器，也可以使用演示模式（实际上，只需要修改快捷方式的参数，后面加`--kiosk`即可）。

### Step 2

离开你的机器，在它更新的时候，没人会处理它！（摸鱼去吧）
